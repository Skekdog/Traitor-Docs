Anyone can make a Traitor! map! You don't need to know how to code to make a map, but if you do, maps can contain almost any custom behaviour, from tasks (like on Skeld), to TVs (like on hidden maps), to complete overhauls of the game.

This guide will assume you have some familiarity with Roblox Studio. If not, I would highly recommend learning that first. You can use the [Creator Docs](https://create.roblox.com/docs) to get started.

To start, you should install the [Traitor! SDK](https://create.roblox.com/store/asset/107938745988414/) plugin. This plugin provides example maps, API types, and the tools you need to successfully load your map in-game.

See also: [`Classes.Map`](/Traitor-Docs/reference/classes/map)
